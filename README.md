
# SMS Spam Detection System

This project implements a machine learning based SMS spam detection system.

## Features
- TF-IDF feature extraction
- Naive Bayes classifier
- Logistic Regression classifier
- ROC-AUC evaluation
- Adversarial robustness testing
- Flask web interface
- Unit testing with pytest

## Project Structure
app/ : Flask web interface
src/ : Machine learning pipeline and training scripts
tests/ : Unit tests
data/ : SMS dataset

## How to Run
pip install -r requirements.txt
python app/app.py
